# 🧩 Universal Web Dashboard (Chrome 扩展)

一个可以将**任意网页的局部区域**剪切并聚合到同一个仪表盘的 Chrome 插件。

## 🚀 安装指南

本插件目前处于**开发者模式**阶段。

1. 克隆 (Clone) 或下载本仓库。
2. 打开 Chrome 浏览器并访问 `chrome://extensions/`。
3. 打开右上角的 **开发者模式 (Developer mode)** 开关。
4. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择包含本项目代码的文件夹。

## 📖 使用说明

1. **添加组件：** 打开任意网页，点击插件图标，选择“开始选择 (Start Selection)”。
2. **编辑模式：** 点击“编辑布局 (Edit Layout)”以启用移动和调整大小功能。
3. **移动组件：** 拖动组件的标题栏即可移动。
4. **调整大小：** 拖动右下角进行调整（支持 10px 自动吸附）。
5. **缩放内容：** 使用组件上的 **+ / -** 按钮缩放网页内容。
6. **撤销操作：** 按 `Ctrl + Z` (或 Mac 上的 `Cmd + Z`) 撤销操作（最多支持 5 步）。
7. **置顶图层：** 点击任意组件即可将其置于最顶层。
8. **删除：** 点击组件标题栏上的 **×** 按钮。
9. **反色/暗黑模式：** 点击 **◑** 按钮可切换该组件的暗黑模式。
